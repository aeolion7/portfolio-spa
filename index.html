BundesData is an application that retrieves soccer data using the user-requested parameters and displays it in an easily understandable and visual format. The user must select which season's data they would like to view, and then, by selecting a matchday and clicking the button, the application will display all of the games that took place on that particular matchday, including team information and logos, half-time and full-time scores, when the match took place (or will take place), and the name (if known) of the head referee who officiated the match.
